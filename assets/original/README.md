# Original Sweet Home OS Pixel Art

This directory holds the first-wave original decoration sprites created for Sweet Home OS.

## Current First-Wave Sprites

- `decor/guild_planter.png` — plant/greenery accent, 16x16 or 32x32 pixels
- `decor/wooden_shelf.png` — shelf/bookcase accent, 16x16 or 32x32 pixels

## Rules

- All sprites must have dimensions divisible by 16.
- Do not import external asset packs here — this directory is for original art only.
- Keep Godot import settings: Nearest filtering, Mipmaps off, Disabled/Lossless compression.
- See `docs/original-sprite-art-handoff.md` for revision guidelines.
